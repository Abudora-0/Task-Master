### Installation


2. Open the application:
   - Simply open `index.html` in your web browser

### Usage

1. **Adding a Task**
   - Enter task name in the input field
   - Select priority level (Low, Medium, High)
   - Choose a due date (optional)
   - Select a category
   - Add notes if needed
   - Click "Add Task" or press Enter

2. **Managing Tasks**
   - Click the checkbox to mark tasks as complete
   - Click the edit icon to modify task details
   - Click the delete icon to remove tasks
   - Drag tasks to reorder them

3. **Filtering & Searching**
   - Use sidebar filters to view specific task categories
   - Use priority filters to focus on important tasks
   - Search for tasks by name in the search box
   - Sort tasks using the dropdown menu

4. **Undo & Clear**
   - Click "Undo" to restore the last deleted task
   - Click "Clear Done" to remove all completed tasks
